# 🚀 Release v1.0 - Dark Mode & Notion-Style TOC

**릴리즈 날짜:** 2026년 1월 20일  
**Git 태그:** `v1.0-stable-toc`  
**상태:** ✅ Stable

---

## 📋 주요 기능

### 1. 다크 모드 (Dark Mode)
- **테마 토글 버튼**: 사이드바 상단에 Sun/Moon 아이콘 버튼 추가
- **CSS 변수 시스템**: 라이트/다크 테마를 CSS 변수로 관리하여 일관성 유지
- **로컬 스토리지 저장**: 사용자가 선택한 테마를 브라우저에 저장하여 재방문 시에도 유지
- **시스템 기본 설정 감지**: `prefers-color-scheme`을 통해 OS 설정 자동 반영
- **부드러운 전환**: 테마 변경 시 0.2초 트랜지션 효과 적용

### 2. Notion 스타일 목차 (Table of Contents)
- **자동 생성**: H1, H2, H3 태그를 파싱하여 자동으로 목차 생성
- **수직 트랙 라인**: 목차 왼쪽에 얇은 가이드 라인으로 구조 시각화
- **계층별 들여쓰기**: H1/H2/H3의 명확한 시각적 구분
- **스크롤 스파이**: 현재 읽고 있는 섹션을 자동으로 하이라이트
- **Sticky 포지셔닝**: 스크롤 시 목차가 화면에 고정
- **반응형 디자인**: 1280px 미만 화면에서는 자동 숨김
- **스마트 표시 로직**: 포스트 읽기 화면에서만 표시, 카테고리 목록에서는 숨김

### 3. 기존 기능 유지
- ✅ 해시태그 시스템
- ✅ 계층형 카테고리 구조 (Posting > Blog, Settings)
- ✅ GitHub Actions 자동 배포
- ✅ 마크다운 기반 포스팅
- ✅ 반응형 디자인

## 🔧 기술 세부사항

### 파일 변경 사항
- `assets/style.css`: 다크 모드 CSS 변수 및 TOC 스타일 추가 (약 80줄 추가)
- `assets/app.js`: 테마 토글 로직, TOC 생성 및 스크롤 스파이 구현 (약 70줄 추가)
- `index.html`: 테마 토글 버튼 및 TOC 사이드바 컨테이너 추가

### 성능 영향
- **번들 크기**: 약 2KB 증가 (압축 전)
- **렌더링 성능**: IntersectionObserver 사용으로 최적화
- **초기 로딩**: 영향 없음 (CSS/JS는 기존 파일에 추가)

## 📦 배포 방법

### 이 버전으로 배포하기
```bash
# 1. 태그로 체크아웃
git checkout v1.0-stable-toc

# 2. GitHub Pages에 자동 배포됨 (GitHub Actions)
```

### 이 버전으로 롤백하기
```bash
# 1. 현재 작업 내용 백업 (선택사항)
git stash

# 2. main 브랜치를 이 버전으로 되돌리기
git checkout main
git reset --hard v1.0-stable-toc

# 3. 강제 푸시 (주의: 이후 커밋이 삭제됩니다!)
git push origin main --force
```
