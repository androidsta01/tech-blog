# Release v1.1

**릴리즈 날짜:** 2026년 1월 26일

Antigravity Tech Blog의 v1.1 업데이트를 배포합니다. 이번 버전에서는 사용자 경험(UX) 개선을 위한 검색 기능과 History UI의 전면적인 개편, 그리고 시스템 안정화에 집중했습니다.

## 🚀 주요 변경 사항

### 1. 강력해진 검색 기능 (Search)
이제 블로그 내의 모든 콘텐츠를 실시간으로 검색할 수 있습니다.
- **실시간 검색**: 타이핑과 동시에 결과가 필터링됩니다.
- **단축키 지원**: `Cmd+K` (Mac) 또는 `Ctrl+K` (Windows)로 언제든지 검색창을 열 수 있습니다.
- **하이라이팅**: 검색어가 포함된 부분을 시각적으로 강조합니다.

### 2. History UI 혁신 (Modernized History)
기존의 정적 텍스트 기반 히스토리 페이지를 완전히 새롭게 디자인했습니다.
- **자동화된 데이터 연동**: `history.json` 데이터를 기반으로 자동 렌더링됩니다.
- **Accordion Grouping**: 날짜별로 그룹화되어 스크롤 압박을 줄이고 가독성을 높였습니다.
- **Modern Card Design**: 카드 스타일 UI와 SVG 아이콘을 적용하여 세련된 느낌을 더했습니다.

### 3. 시스템 안정화 및 개선 (Improvements)
- **Tag 파싱 오류 수정**: 태그에 불필요한 대괄호(`[]`)가 표시되던 문제를 해결했습니다.
- **다크 모드 개선**: 일부 요소의 가시성 문제를 해결했습니다.
- **콘텐츠 확장**: 최신 기술 트렌드를 다루는 `Trend` 카테고리를 신설하고 첫 포스트를 발행했습니다.

## 📝 향후 계획
다음 버전(v1.2)에서는 SEO 최적화 및 모바일 뷰 개선에 주력할 예정입니다.
사용자 여러분의 피드백은 언제나 환영입니다.
